### 👋 About Me

I'm a **Senior Software Engineer** at **BRAC IT Services Ltd.**, Dhaka, with **5+ years** of experience architecting and delivering enterprise-grade backend systems across **banking, fintech, LMS, and e-commerce** domains.

My work powers platforms used by **BRAC Bank**, **bKash**, and **British American Tobacco** — systems that handle real transactions, real users, and zero tolerance for downtime.

I don't just write code. I design systems that teams can **own, extend, and trust in production** — with a focus on clean architecture, API reliability, and sustainable engineering practices. When I'm not building, I'm mentoring — because the best systems are built by **leveled-up teams**, not lone heroes.
